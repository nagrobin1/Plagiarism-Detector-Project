How to run Plagiarism Detector

  • 1) Click on Get to Work! button or Login link in the navigation bar
  • 2) Enter your credentials. For testing use: username = Tester, password = Test123
  • 3) Instructor needs to upload only ONE zip file which will contain repositories of students. Only java files in the student's submission will be considered by the application and any other files will be ignored.

  • 4) Example of expected zip file format:

    Please only upload submission.zip file.

  • 5) Once files are uploaded successfully, click Detect Plagiarism button.
  • 6) Results will be loaded in the same dashboard page and you can view more details about any perticulr match by clicking Show More Details button.
